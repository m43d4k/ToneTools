# ToneTools

Tools for understanding and shaping sound.

## Overview
ToneTools is a small collection of web tools related to sound and music. 

## Tools
- [Note ↔ Frequency](https://m43d4k.github.io/ToneTools/note-frequency/)  
— note–frequency table (MIDI C3 = 60 convention)

- [Transpose](https://m43d4k.github.io/ToneTools/transpose/)  
— calculate semitone and interval between notes

- [Phase → Delay Converter](https://m43d4k.github.io/ToneTools/phase-delay/)  
— Convert phase shift (degrees) to time delay or advance in ms and samples.

- [Delay & Reverb Calculator](https://m43d4k.github.io/ToneTools/delay-reverb/)  
— Calculate pre-delay, decay, and total reverb time from BPM
